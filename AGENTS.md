# Agents Guide: Capstone Analisis Kelas Paralel

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Key Libraries:** Pandas, Plotly, SciPy, OpenPyXL

## Development Commands
- **Run App:** `streamlit run analisis_kelas_paralel.py`

## Project Architecture
- **Entrypoint:** `analisis_kelas_paralel.py` - Contains the entire dashboard logic including data cleaning, ANOVA analysis, and visualization.

## Data Requirements
- **Input Format:** Excel (`.xlsx`)
- **Required Columns:** `th_ajaran`, `kode_makul`, `grup`, `nilai_angka`, `dosen`, `kode_prodi`
